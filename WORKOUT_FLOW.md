# Workout Flow — الخوارزمية وفلو العمل الكامل

**Version:** 2.0 (wizard flow)
**المرجع:** طلب المستخدم 2026-07-15 + spec v2 §4.5 (البرنامج هيكل فقط، الأوزان بالجلسة)

---

## 1. المبدأ الأساسي (غير قابل للكسر)

| المكان | ماذا يُخزَّن | ماذا يُمنَع |
|---|---|---|
| البرنامج (القالب) | اسم التمرين، النوع، العضلة، عدد الدفعات، **Target reps لكل دفعة (اختياري)** | أي وزن — الوزن لا يُخطط أبداً |
| الجلسة (Active Workout) | الوزن والعدّات الفعلية لكل دفعة | تعديل القالب |

- **Target reps** يأتي من البرنامج ويُعبَّأ مسبقاً في حقل العدّات (قابل للتعديل).
- **حقل الوزن يبدأ فارغاً دائماً** — المستخدم يسجله وقت التمرين.

## 2. فلو الشاشة (Wizard)

```
Start Workout
  └─> شاشة التمرين النشط تفتح مباشرة على أول تمرين — لا قائمة فارغة
       ┌──────────────────────────────────────┐
       │ Header: اسم اليوم · التاريخ           │
       │ تمرين i من N + شريط تقدم              │
       ├──────────────────────────────────────┤
       │ التمرين الحالي (مفتوح):               │
       │  • الاسم · العضلة · عدد الدفعات       │
       │  • آخر مرة لعبت (كل دفعة)             │
       │  • أعلى وزن (+ تاريخ آخر تحقيق)       │
       │  • Today: لكل دفعة سطر:               │
       │      Set n · Target: X reps           │
       │      Weight [فارغ] · Reps [=Target]   │
       │  • + Add Set   • حذف دفعة             │
       │  • [Complete Exercise]  ← الزر الأساسي │
       ├──────────────────────────────────────┤
       │ Up Next: باقي التمارين مصغّرة          │
       │ (المكتمل ✓ أخضر — الضغط يعيد فتحه)    │
       │ Finish Workout: رابط ثانوي صغير فقط    │
       └──────────────────────────────────────┘
Complete Exercise (للتمرين الأخير غير المكتمل)
  └─> كل التمارين مكتملة → الزر الأساسي يصبح [Finish Workout]
       └─> تقرير نهاية اليوم (كل تمرين: دفعاته وأوزانه وعدّاته)
```

## 3. الخوارزميات

### 3.1 اختيار التمرين المعروض
```
isDone(exercise) = exercise.sets غير فارغة AND كل دفعاتها is_completed
openExercise     = المستخدم اختار تمريناً يدوياً من Up Next؟ ذلك التمرين
                   وإلا: أول تمرين isDone(e) == false
                   وإلا (الكل مكتمل): لا شيء مفتوح — تظهر بطاقة "خلصت كل التمارين"
progress         = عدد المكتملة / العدد الكلي   (الهيدر: "تمرين (مكتملة+1) من N")
```

### 3.2 بدء الجلسة (startWorkout)
```
لكل تمرين في اليوم، لكل دفعة j من 1..defaultSets:
  weight = NULL          ← دائماً، لا استثناء
  reps   = targetReps[j] ← إن وُجد هدف بالبرنامج، وإلا NULL
  is_completed = 0
استئناف جلسة موجودة: نفس الشاشة، تفتح على أول تمرين غير مكتمل
```

### 3.3 Complete Exercise
```
عند الضغط:
  validSet(s) = s.reps != null AND (النوع reps-only OR s.weight != null)
  1. إذا لا توجد أي دفعة valid → خطأ: "سجّل دفعة واحدة على الأقل" — لا انتقال
  2. كل دفعة valid  → is_completed = 1
  3. كل دفعة فارغة → تُحذف (وإعادة ترقيم 1..n)   ← "إلغي دفعة إذا تعبت"
  4. التمرين يصبح isDone → ✓ أخضر في Up Next
  5. الانتقال تلقائياً لأول تمرين غير مكتمل تالٍ (إعادة تعيين الاختيار اليدوي)
```

### 3.4 Finish Workout
```
- أثناء وجود تمارين غير مكتملة: Complete Exercise هو الأساسي؛
  Finish Workout رابط ثانوي صغير (لمن يريد الإنهاء المبكر)
- بعد اكتمال كل التمارين: Finish Workout يصبح الزر الأساسي الوحيد
- الشرط القائم: دفعة مكتملة واحدة على الأقل بالجلسة كاملة
- النتيجة: تقرير نهاية اليوم + تحديث "آخر مرة لعبت" و"أعلى وزن"
```

### 3.5 Target reps في البرنامج (نموذج البيانات)
```
جدول workout_day_exercise_sets (موجود من schema v2):
  workout_day_exercise_id, set_number, target_reps
  target_weight يبقى NULL دائماً (§4.5)
نموذج الفورم: صف لكل دفعة: "دفعة i — Target reps [اختياري]"
  + إضافة/حذف دفعات (1..20). عدد الدفعات = عدد الصفوف.
```

## 4. حالات حدّية

| الحالة | السلوك |
|---|---|
| لا توجد جلسة سابقة | "لا توجد بيانات سابقة بعد" في قسم آخر مرة لعبت |
| Complete بدون أي قيمة | رسالة خطأ محلية، يبقى التمرين مفتوحاً |
| دفعات فارغة عند الإكمال | تُحذف ويُعاد الترقيم |
| Add Set بعد الإكمال | يعيد التمرين لغير مكتمل (يفتح مجدداً) |
| إغلاق التطبيق أثناء الجلسة | الاستئناف يفتح أول تمرين غير مكتمل |
| الضغط على تمرين مكتمل في Up Next | يفتح للتعديل، Complete يعيد حسابه |
| تمرين reps-only | لا حقل وزن إطلاقاً؛ Target والقيم عدّات فقط |
| Finish مبكر مع تمارين ناقصة | مسموح (رابط ثانوي)، بشرط دفعة مكتملة واحدة |
